Employment Availabilities

Job Specification

Position: Junior Front End Web Developer


Type: Full-time


Location: A Place


Salary: £40,000 - £45,000


Details:
As a front-end web developer, you would be in charge of the looks of the website. This will involve meetings with the client and design teams to ensure that the website matches and looks like what the client wants. Additionally, you would be in charge of creating responsive designs ensuring usability across devices of all types making use of HTML, CSS and JS.



Position: Junior Back End Web Developer


Type: Full-time


Location: A Place


Salary: £55,000 - £75,000


Details:
As a back-end web developer it is your role to create user interactive features and control the server side of the website to allow users to engage with the website and to bring the clients want to life in terms of accounts and server management, this being done with help from server side scripting languages such as PHP and some forms of JS.


Qualifications

There are no formal qualifications needed for our web development roles with you starting out as a junior member of our team, however basic qualifications in maths and English as well as additional qualification in web development subjects such as computer science, IT, or Web Design will be beneficial.

For these roles however you will need to know the following programming languages:.

HyperText Markup Language - HTML

Cascading Style Sheets - CSS

JavaScript - JS

Java

Hypertext Pre-processor - PHP

Python
